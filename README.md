# VIM-POSTIT
This is vim plugin. NOT for neovim.
You can PEEK your code, and handle like Post-It.

# Instllation
This plugin can install by same way of other general plugins.

# Usage
Peek selected lines
```vim
:'<,'>Peek
```

Handle code-block
```vim
:MoveStart
```

## Keybinding
| operation | key                          |
|-----------|------------------------------|
| Peek      | <kbd>Ctrl<kbd> + <kbd>p<kbd> |
| MoveStart | <kbd>Ctrl<kbd> + <kbd>m<kbd> |
| left      | <kbd>h<kbd>                  |
| down      | <kbd>j<kbd>                  |
| up        | <kbd>k<kbd>                  |
| right     | <kbd>l<kbd>                  |
